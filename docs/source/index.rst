Welcome to Foot-Work blender addon documentation!
===================================

**Foot-Work** is collection of tools to work with locomotion animation in Blender.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   The development is done on Blender 4.2 LTS.  
   The Addon might work on other versions but there's no guarantee for now.
